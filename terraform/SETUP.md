# Terraform Setup

## Prerequisites
- Terraform installed
- AWS credentials configured

## Steps
1. Initialize Terraform:
   ```sh
   cd terraform
   terraform init
   ```
2. Apply infrastructure:
   ```sh
   terraform apply
   ```
3. Review outputs for created namespaces and resources.
4. Use outputs to configure your Kubernetes and CI/CD setup.
